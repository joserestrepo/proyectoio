Instrucciones: Cómo usar la herramienta SXR

A continuación podrá leer una breve y sencilla guía de uso para la herramienta SXR. Una vez que haya modelado el problema de programación lineal, es decir, tenga identificada la función objetivo a maximizar/minimizar junto con sus restricciones, puede estar seguro de que la tarea más difícil ha terminado. Deje ahora que SXR haga el trabajo.



Para esto utilizaremos el siguiente ejemplo:
Maximizar Z = f(x,y) = 3x + 2y
sujeto a: 2x + y ≤ 18
2x + 3y ≤ 42
3x + y ≤ 24
x ≥ 0 , y ≥ 0

Lo primero que debemos hacer es ingresar los valores correspondientes al número de variables de decisión y numero de restricciones.


Al ingresar el número de variables de decisión, automáticamente se despliega un campo para ingresar la función objetivo y otro para indicar si se desea maximizar o minimizar


Al ingresar el número de restricciones, automáticamente se despliegan los campos respectivos para cada restricción.


Por ultimo presionamos el botón igual para obtener el resultado.
